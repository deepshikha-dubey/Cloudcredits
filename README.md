## Project-3: Creating A Chatbot
# 🤖 AI Chatbot using Flask, Docker & CI/CD
This project is a simple chatbot application built using Flask and Docker.
The chatbot can respond to user queries and is designed to demonstrate DevOps practices like containerization and deployment.

FEATURES
- Chatbot UI using HTML, CSS, Bootstrap
- Backend using Flask
- Docker containerization
- Basic chatbot logic / AI integration
- Ready for deployment

How to Run Locally
### Step 1: Clone repo
git clone https://github.com/yourusername/project-3-creating-chatbot.git

### Step 2: Build Docker image
docker build -t chatbot-aap .

### Step 3: Run container
docker run -p 5000:5000 chatbot-aap

### Step 4: Open browser
http://localhost:5000

DOCKER COMMANDS
 - docker build -t chatbot-aap .
 - docker run -p 5000:5000 chatbot-aap
 - docker ps
